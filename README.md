# Atividade Prática

Este projeto foi desenvolvido como parte da atividade prática de da disciplina de Programação Orientada a Objetos do Curso Superior de Tecnologia em Análise e Desenvolvimento de Sistemas (IFSC - Campus São José)

O objetivo é implementar um sistema de gerenciamento de custos de cursos, aplicando os pilares da Programação Orientada a Objetos (POO): **Encapsulamento**, **Herança**, **Polimorfismo** e **Associação**
